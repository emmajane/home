Grunt is a task runner

written in JavaScript (Node)

What can I do with it?

  • Minify & concatentate JS files
  • Compile Sass or LESS styles
  • Run tests
  • Lint JS & CSS files
  • Optimize images
  • Prepare a build & deploy it

Okay, it can do a lot...

Actually, it doesn't do much at all.

Grunt is only a task runner

Not a set of executable tasks

Instead, Grunt has a long list of task plugins... over 4,000.

So what is it?

Grunt is a series of Node modules.

My first time

How to get running with Grunt.

  • Step 1: Install Node.js

  • Step 2: Install the Grunt CLI

    
~/my-project$ npm install -g grunt-cli

  • Step 3: Create package.json and Gruntfile.js
    (in your project root, most likely)

  • Step 4: Configure and run some tasks!
